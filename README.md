# Phone Book Search
This program identifies if different search querries are present in a large directory like a phone book. 

## Source code
task/src/phonebook/Main.java

## About
This program is build basted on a project-based learning project from Hyperskill. Implementing different sort and search algorithms in order to identify if an entry of a list (find.txt)
is present in a phonebook (directory.txt). Keeping track of time to compare different methods to compare the effectiveness of earch algorithm.
